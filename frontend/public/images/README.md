# Image Assets

Each PNG in this folder is a single-pixel color swatch that is reused across the project.

| File | Description |
| --- | --- |
| `linen.png` | Warm ivory base, used for hero and product backdrops. |
| `terra.png` | Earthy terracotta accent used for spa and kitchen scenes. |
| `sage.png` | Soft green nodding to foliage-forward styling. |
| `apricot.png` | Golden apricot to highlight cozy textiles. |
| `sand.png` | Neutral sand tone for support/product spotlights. |
| `storm.png` | Moody slate for contrast in galleries. |
| `forest.png` | Deep forest tone for the narrative/hero storytelling panels. |
| `cerulean.png` | Fresh cerulean for lookbook and living moods. |
| `sunset.png` | Warm sunset for spotlight cards. |
| `dusk.png` | Muted twilight for testimonials and navigation mood overlays. |
